# Changelog

All notable changes to LibVanced will be documented in this file.

## [1.0.0] - 2026-03-20

### Initial Release
- Hierarchical tree view (Artist > Album > Track)
- Configurable grouping patterns via titleformat
- Album art thumbnails for group nodes
- Drag & drop to playlists and queue (SimPlaylist/Queue Manager interop)
- Context menu with Send to Playlist, Add to Queue, and full SDK context menu
- Keyboard shortcuts (Q to queue, Enter to send to playlist, Space to pause)
- Search/filter bar with media library search syntax
- Library change callbacks for live updates
- Glass (transparent) background support
- Track count badges on group nodes
